国战部 ( Country And War )

国战部（Country And War），简称CAW，成立于2017年4月23日，是一个依托Minecraft平台、QQ群的组织的综合性游戏组织。

截至2023年7月，国战部共有成员60位，已经开展了数十次MC、文游活动，其中著名的有
国战5.0OL、国战9.0、国战服务器7235402、1652484等、第一次文游、2021春季文游、2021暑假服务器、2021年暑假文游
等。

截至2022年7月，国战部已经开启了11次网易MC服务器，10次网易MC房间、13次文游，总接受捐款42元，总花费198元。

国战部信息系统 ( C.A.W. Infomation System )

国战部信息系统 ( C.A.W. Infomation System ) 简称 “国战信息” ( C.I.S ) ，隶属于内务部网络与安全司 ( N.S.D. - M.I.A. ) ，前身为国战部信息库 ( C.A.W. Infomation Storage ) ，为国战部提供独立信息发布渠道，并归档国战部历史资料与文件（包括文游、Minecraft服务器等）。

当今版本 ‘Aris’ 24.1 基于Hexo搭建。
